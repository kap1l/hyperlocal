# microWeather

Hyperlocal weather for runners.

## Manual Setup (Node.js Missing in Environment)
It appears Node.js and npm were not found in your current environment PATH.

To run this project:
1. Ensure Node.js is installed.
2. Open a terminal where `node` and `npm` are accessible.
3. Run `npm install` to install dependencies.
4. Run `npx expo start` to launch.

## Features
- **Home**: Minute-by-minute rain chart, safe run window indicator.
- **Radar**: overlaid precipitation map.
- **Hourly**: 48h forecast.
- **Alerts**: Custom rain start alerts.
- **Settings**: API Key configuration for Pirate Weather.

## API Key
You need a Pirate Weather API Key. Enter it in the Settings screen upon launch.
